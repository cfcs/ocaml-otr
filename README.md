OCaml-OTR
=========

[Off-the-record (OTR)](https://otr.cypherpunks.ca/) messaging protocol, purely in OCaml.

Only versions 2 and 3 are supported.

Best to be used with [jacline](http://github.com/hannesm/jackine).
